# Text and LaTeX Notes

## Practical Selection Guide

- `Text` for prose and UI-style labels.
- `MarkupText` for limited inline emphasis.
- `Paragraph` for multi-line blocks that should stack naturally.
- `Tex` for LaTeX-formatted text and short explanations.
- `MathTex` for equations, identities, and derivations.

## Layout Patterns

- title + subtitle
- formula + explanation
- derivation with one active line
- graph label + annotation

## Debugging Hints

- Increase font size before restructuring the whole scene.
- Simplify failed math expressions and isolate the problematic fragment.
- Move the diagram first, then the label.
- Split dense text into multiple beats or scenes.
