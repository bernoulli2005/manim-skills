# Animation Patterns

## Choose The Simplest Effective Animation

- `Create` for drawing an object into existence.
- `FadeIn` and `FadeOut` for presence changes.
- `Write` for text and formulas.
- `Transform` for conceptual continuity.
- `ReplacementTransform` for a clean handoff.

## Choreography Rules

- Combine motions that belong to the same beat.
- Separate beats when the viewer needs to understand each step.
- Use `run_time` to control readability.
- Use staggered reveals for lists and repeated structures.

## Avoid

- Overusing special effects.
- Using complicated motion when a simpler one would teach better.
- Chaining too many animation types in one visual beat.
