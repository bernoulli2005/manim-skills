# Text and LaTeX

## Use The Right Text Class

- `Text` for ordinary prose.
- `MarkupText` for styled spans.
- `Paragraph` for multi-line text blocks.
- `Tex` for LaTeX-rendered text.
- `MathTex` for symbolic math.

## Rules

- Keep text readable at render scale.
- Use LaTeX only when typesetting precision matters.
- Keep labels short near dense diagrams.
- Stage derivations instead of crowding the frame.

## Common Mistakes

- Using `MathTex` for everything.
- Making labels too small.
- Overlapping text with geometry.
- Mixing too many text styles without hierarchy.
