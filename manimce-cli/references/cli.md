# CLI Notes

## Good Render Loop

1. Render with a low-quality preset.
2. Check motion, timing, and layout.
3. Increase quality only after the scene is correct.

## Debugging Checklist

- scene class name exists
- file path is correct
- `manim.cfg` is not overriding your flags
- output folder is the one you expect
- stale media is not confusing the result

## Practical Habit

- Keep a local config for stable defaults in a project.
